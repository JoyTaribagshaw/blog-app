## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:
  * A basic understanding of Ruby and Git
### Usage
1. Install Ruby
2. Install Rails
3. Clone this repo 
```
git clone git@github.com:JoyTaribagshaw/blog-app.git
cd table_set_up
```
```
run
rails server
```

### Run RSpec Tests

```
 rspec spec
```
